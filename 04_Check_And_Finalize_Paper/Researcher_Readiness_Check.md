# Researcher Readiness Check

Use this file before starting a real project.

## Can you start with only a rough idea?

Yes. You can start if you know at least:

```text
field or topic
why it might matter
whether you will use Codex or Claude Code
what materials are safe to share
```

PaperPilot can help turn that into a scored idea and question options.

## Do you need to read internal skills?

No. The skills are for the agent. You can start from:

```text
01_Start_Here/README.md
01_Start_Here/Use_Codex.md
01_Start_Here/Use_Claude_Code.md
01_Start_Here/Run_Full_Paper_Process.md
```

Read or tailor the internal skills only if you want to customize the workflow.

## Should you use PaperPilot directly or tailor it?

| Situation | Recommendation |
| --- | --- |
| You want to test the workflow | Use directly |
| You have a specific dataset/license setup | Tailor data rules first |
| You have coauthors or lab rules | Tailor decision and evidence rules |
| You target a specific journal/conference | Tailor quality score and writing rules |
| You work with restricted data | Tailor data safety before using any agent |

## Are you ready for a full paper run?

You are ready when:

- you selected Codex or Claude Code;
- the data-safety note has been read;
- the agent can start in Plan Mode;
- you are willing to make decisions through decision packets;
- you will let the required evidence records update in parallel.
