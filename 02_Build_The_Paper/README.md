# 02 Build The Paper

Use this folder for the actual paper work.

## Folders

| Folder | Use it for |
| --- | --- |
| `01_Question_And_Story/` | Research idea, question, contribution, mechanism, and paper narrative. |
| `02_Data_And_Code/` | Safe data notes, metadata, toy data, code, and analysis plans. Do not store restricted or licensed raw data unless allowed. |
| `03_Results_And_Manuscript/` | Tables, figures, result summaries, manuscript drafts, appendices, and slides. |

## Agent context

Keep [`PAPER_CONTEXT.md`](PAPER_CONTEXT.md) updated. It tells Codex or Claude Code what the paper is about, what is already decided, and what must not be assumed.
